#### 5MinutesCountdownMod 1.0.0 for Minecraft 1.12.2 ####

# 前提Ver&Mod

・Minecraft1.12.2+
・MincraftForge 1.12.2_14.23.5.2847+

# 使用方法

・MincraftForgeの導入後に作成されるmodsフォルダの中に、DLしたファイルを.jarファイルのまま置いてください。
・クライアント・サーバーの両方に導入する必要があります。


# 配布DL先
 Wiki：https://defeatedcrow.jp/modwiki/5MinutesCountdown


# 概要
　プレイヤーが死亡してからデスポーン地点で5分間が経過するまでをカウントダウン表示してくれるmodです。

・デスポーン地点から128ブロック以上離れている間はカウントダウンを停止します。
・5分間経過するか、プレイヤーがデスポーン地点に近づくとカウントダウンを終了します。
・デスポーン地点にスティーブ頭が出現し、遠くからでも見つけやすくなります。(コンフィグでオフにできます。）

※タイマーの位置や長さはコンフィグで調整できます。
